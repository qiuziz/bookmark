# 书签管理工具

一个基于 React 和 TypeScript 构建的现代化、响应式书签管理应用。

## 项目简介

书签管理工具是一个网页应用，旨在帮助您高效地组织、管理和导入/导出书签。它提供简洁、响应式的界面，支持层级文件夹结构和 Edge 浏览器书签文件的导入导出功能，让您能够直观地浏览和整理书签收藏。

[English Introduction](./README.md)

## 为什么需要这个项目

在不同浏览器和设备间管理书签可能具有挑战性。许多用户随着时间积累大量书签，但缺乏有效的工具来组织它们。创建这个项目是为了提供：

- 统一的书签管理界面，支持不同来源的书签
- 支持层级文件夹结构，类似传统的书签管理器
- 便捷的导入导出功能，便于在不同浏览器间迁移
- 现代化、响应式设计，在桌面和移动设备上都能良好运行
- 置顶重要书签以便快速访问

## 功能特性

- **层级文件夹管理**：将书签组织成嵌套的文件夹结构
- **响应式设计**：简洁现代的界面，可适应不同屏幕尺寸
- **置顶重要书签**：高亮和优先显示您最重要的书签
- **导入/导出**：从 Edge 浏览器 HTML 导出文件导入书签，导出您的收藏
- **添加/编辑/删除**：完整的书签管理操作
- **基于 URL 的导航**：通过 URL 分享特定文件夹的深度链接

## 如何运行项目

### 前置要求

- Node.js 18+ 
- pnpm（推荐）或 npm

### 安装和运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build

# 预览生产版本
pnpm run preview
```

### 环境配置

项目使用 Vite 作为构建工具，基本使用无需额外配置环境变量。
